# Tasks Generator

A simple full-stack web app that generates user stories and engineering tasks from a feature idea.

## Features
- Input form for goal, users, and constraints
- Generate user stories and tasks
- Editable tasks
- Export as Markdown
- Last 5 specs
- Template selector
- Risks/Unknowns section
- Status page

## Tech Stack
- Frontend: React (Vite)
- Backend: Node.js, Express
- Storage: JSON

## Live App
https://tasks-generator-theta.vercel.app

## How to Run

### Backend

cd backend
npm install
node server.js

### Frontend

cd frontend
npm install
npm run dev
